# ESPHome Components

Additional ESPHome external components maintained by Barna Szternák.

This repository contains custom components developed for ESPHome. They are designed to behave like native ESPHome components and can be installed directly from GitHub using the `external_components` feature.

## Available components

| Component    | Description                      | Status    |
|--------------|----------------------------------|-----------|
| **VEML6040** | Vishay RGBW ambient light sensor | ✅ Stable |

## Installation

```yaml
external_components:
  - source: github://kzd76/esphome-components
```

Then use the component as described in its documentation.
